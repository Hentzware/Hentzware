### TalkCorner

<img src="TalkCorner/TalkCorner.png" alt="TalkCorner" width="600">

<br>

Modernes, selbst gehostetes Forum — gebaut mit **Angular 21** und **ASP.NET Core 10** in Clean Architecture.
All-in-One-Plattform ohne Plugins: Boards, Threads, Private Nachrichten mit Ende-zu-Ende-Verschlüsselung,
Echtzeit-Features via SignalR, Admin-Panel, OAuth, 2FA, Dark/Light Theme und Internationalisierung.
Deployment per `docker compose up`.

### HyperForge

Hyper-V Lab Builder mit GUI — erstelle komplette Lab-Umgebungen (Active Directory, DNS, DHCP, Windows Server, Clients)
per Mausklick, ohne PowerShell-Kenntnisse. Zielgruppe: IT-Azubis, Umschüler, Studierende und Homelab-Enthusiasten.
Mit ISO-Management, Answer File Builder, Netzwerk-Topologie-Visualisierung, Golden Images und Lab Templates.
